# Roadmap

## Phase 1: macOS Prototype

- Region selection overlay.
- Manual scroll target selection.
- Native screenshot capture.
- Automated scroll loop.
- Basic overlap stitching.
- PNG export.

## Phase 2: Reliability

- Stitch confidence scoring.
- Sticky header/footer masking.
- Manual correction UI.
- Capture preview.
- Maximum length and timeout controls.
- Better handling for virtualized lists and chats.

## Phase 3: Desktop App

- Menu bar capture shortcut.
- Capture history.
- Export presets.
- Clipboard integration.
- Permission onboarding.

## Phase 4: Cross-Platform Support

- Windows capture and UI Automation adapter.
- Linux X11/Wayland research spike.
- Shared stitching engine.

## Phase 5: Advanced Capture

- Upward capture for older chat history.
- Bidirectional capture.
- PDF export.
- Batch capture.
- Redaction tools.
